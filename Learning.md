from a concept to a live, deployed web app.

  workflows, and automated deployment via GitHub Actions
- **Web development** — HTML, CSS, and JavaScript; responsive and mobile-first
  layouts; working with mapping libraries and external APIs
- **UI/UX design** — layout, visual hierarchy, and Soldier-centered workflows
- **Data modeling** — star schema, snapshot warehousing, ETL, and DAX in Power BI
- **Software architecture** — structuring maintainable projects and shipping
  iteratively based on user feedback
