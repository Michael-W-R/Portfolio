# UI/UX Design Portfolio

This section highlights my approach to designing Soldier-centered digital tools. My UI/UX work focuses on clarity, simplicity, and reducing friction in workflows that traditionally rely on manual processes or outdated systems.

Because many Army systems require CAC access, the examples here are non-sensitive recreations, workflow diagrams, and design rationale that demonstrate how I translate operational requirements into intuitive interfaces.
