# Army Projects — Case Studies

Non-sensitive summaries of the HR-modernization and readiness tools I've
designed and built. Each case study covers the problem, my solution and role,
the impact, and the skills it demonstrates for the Army Software Factory (ASWF).
All examples use non-sensitive sample data; nothing here contains CUI or
CAC-restricted content.

## Flagship technical builds
- **[Senior Rater Profile Manager](Senior-Rater-Profile-Manager.md)** — a Power BI
  rule engine that encodes AR 623-3 / DA PAM 623-3 evaluation-profile logic and
  lets senior raters model planned ratings before they misfire a profile.
- **[Strength Management Tool](Strength-Management-Tool.md)** — a snapshot
  star-schema Power BI suite that consolidates personnel-system exports into
  authorized-vs-assigned strength, gains/losses, and forward force projection.

## Product & workflow case studies
- **[Rating Chain App](Rating-Chain-App.md)** — a Soldier-facing tool for

- **[Evaluation Automation Tools](Evaluation-Automation.md)** — automation across

- **[Workflow Optimization & Data Integration](WorkFlow-Optimization.md)** — HR
  process modernization through integrated data pipelines and automated workflows.
- **[Peer Evaluation Tool](Peer-Evaluation-Tool.md)** — a lightweight, accessible
