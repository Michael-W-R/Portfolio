# Evaluation Automation Tools — Case Study

## Overview
I developed multiple automation tools that streamline the Army evaluation process, reduce administrative workload, and improve accuracy across G‑1 operations. These tools support both Soldier-facing and staff-facing workflows.

## Problem
Evaluation processing is time-consuming, error-prone, and heavily dependent on manual data entry. Units often struggle with policy interpretation, suspense tracking, and ensuring evaluations move through the correct routing chain.

## Solution
My automation tools provide:
- Automated routing logic based on rank, component, and duty status  
- Suspense tracking dashboards  
- Error-checking against AR 623-3 requirements  
- Workflow automation using Power Apps, Power BI, and SharePoint  
- Clear visualizations for commanders and HR staff  

## My Role
I built the data models, designed the workflows, and created the dashboards and automation logic. I worked directly with staff sections to identify pain points and iterated based on feedback.

## Impact
- Reduced processing time  
- Improved accuracy and compliance  
- Increased transparency for commanders  
- Better workload distribution across HR sections
